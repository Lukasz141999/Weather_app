# Weather Flask App

Simple weather application built with:

- Python
- Flask
- PostgreSQL
- Docker
- OpenWeather API


## Tech stack
- Flask
- PostgreSQL
- Docker
- HTML/CSS

## Features

- Select city and fetch weather
- Save weather to PostgreSQL database
- View weather history
- Fully containerized with Docker

## Run project

```bash
docker compose up --build
```
Open in browser:
http://localhost:5000
